## ⚡ The Edge-Native Utility Hub

NadirTools is a modern, privacy-first platform designed to replace ad-heavy, slow, server-side calculators. Every tool in this suite executes **100% client-side**, meaning your network configurations, firewall rules, and financial numbers *never leave your browser*.

> **Launch the Platform:** [https://nadirtools.com](https://nadirtools.com)

### 🛠 Core Engines Currently Live

*   **Subnet & CIDR Partition Engine**: Visually slice and calculate IPv4 network blocks dynamically.
*   **Firewall Rule Validator**: Parse and audit OPNsense and Cisco ACL rules for shadowing and logic errors.
*   **JSON Parser & Schema Formatter**: Validate, minify, and pretty-print JSON payloads with real-time syntax tracing.
*   **Cron Schedule Visualizer**: Decode complex Cron expressions into human-readable text and simulate future execution times.
*   **Tech Freelance Tax Matrix**: Calculate net take-home pay, social taxes, and VAT/GST thresholds across the US, UK, and AU.
*   **Cash-on-Cash Yield Simulator**: Model commercial real estate amortization, NOI, and Debt Service Coverage Ratios (DSCR).

## 🚀 Built For Speed & Scale
Engineered inside a Turborepo monorepo with **Next.js 15** and **React 19**, and deployed seamlessly to **Cloudflare Pages**.

*   **Zero Latency**: No server API round-trips. Instant math via native JavaScript.
*   **Privacy Absolute**: No tracking databases, no payload collection.

## 🤝 Contributing
NadirTools is an open-source initiative aiming to clean up the developer utility ecosystem. Pull requests for new mathematical engines are highly encouraged!

### Local Development
```bash
# Clone the repository
git clone https://github.com/casp3rgr33n/nadirtools.git

# Navigate to the web application
cd nadirtools/apps/web

# Install dependencies
npm install

# Start the local development server
npm run dev
```
