### ⚡ The Edge-Native Engine
NadirTools is a modern, privacy-first platform designed to replace ad-heavy, latency-ridden server calculators. Every engine in this suite executes **100% client-side**. Your network configurations, firewall rules, and financial numbers *never leave your browser*.

<br />

### 🛠 Core Modules
*   **Subnet Partition Engine**: Visually slice and calculate IPv4 network blocks dynamically.
*   **Firewall Rule Validator**: Parse and audit OPNsense and Cisco ACL rules for shadowing and logic errors.
*   **JSON Schema Formatter**: Validate, minify, and pretty-print JSON payloads with real-time syntax tracing.
*   **Cron Schedule Decoder**: Translate complex Cron expressions into human-readable text and simulate execution triggers.
*   **Tech Freelance Tax Matrix**: Calculate net take-home pay, social taxes, and VAT/GST thresholds.
*   **Cash-on-Cash Yield Simulator**: Model commercial real estate amortization, NOI, and Debt Service Coverage Ratios.

<br />

### 🚀 Architecture
Engineered inside a Turborepo monorepo with **Next.js 15** and **React 19**, deployed directly to the **Cloudflare Global Edge**.

*   **Zero Latency**: No server API round-trips. Instant mathematical resolution via native JavaScript.
*   **Privacy Absolute**: No tracking databases, no payload collection, zero external telemetry.

<br />

### 🤝 Local Execution
NadirTools is an open-source initiative. To audit or run the mathematical engines locally:

```bash
git clone https://github.com/casp3rgr33n/nadirtools.git
cd nadirtools/apps/web
npm install
npm run dev
```
